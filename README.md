# SwiftOrder
> ***Kiosk Ordering Made Easy***

SwiftOrder is a web-based kiosk ordering application.

<br>

## Built with
### Back End
- C#
- .NET 6
- LINQ
- Entity Framework

### Front End
- JavaScript
- React
- HTML/CSS
- SASS
